Instructions for installing the REV into “BibleWorks” on a Windows PC

It is assumed you have Bibleworks 6 or greater. This -may- work with earlier versions, but it has not been tested.
  1. You must have BibleWorks installed on your PC.
  2. Make sure that BibleWorks is not running.
  3. Download the REV_Bibleworks.zip from the Export section of the REV website, keeping track of where your browser is saving the file to. Usually, this will be in your “Downloads” folder, as seen here in Windows Explorer.
  4. Copy or move the rev.ddf file into the folder "C:\Program Files (x86)\BibleWorks XX\userdb". Replace the XX with the version number of the BW program you own.
  5. Copy or move the rev.txt file into the folder "C:\Program Files (x86)\BibleWorks XX\databases".
  6. Start Bibleworks.
  7. Click "Tools/Importing/Exporting Information/Version Database Compiler".
  8. In the window that opens, under #1 "Database Description File (DDF)", click "Open", and load the rev.ddf file.
  9. Under #2 "Database Raw Text File", click "Browse" and navigate to C:\Program Files (x86)\BibleWorks XX\databases, and double-click the "REV.txt" file that you just copied there in step 5.
  10. In the lower right, make sure "Install after Compiling" is checked, and then click the "Compile" button.
  11. The database should compile, and you should see "...Build was successful. Installed in: c:\program files (x86)\bibleworks XX\databases" in the output window.
  12. Click the "Close" button, and the REV should now be available from within Bibleworks. Note that if you are using BibleWorks version 10 you will be prompted to restart the program. Go ahead and do so.
Due to a limitation of the way we are importing the REV into BibleWorks, Greek and Hebrew words have been removed from the commentary, and diacritic letters have been converted to standard ASCII characters.
